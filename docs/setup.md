# Setup

## Web

1. Install `pnpm`.
2. Run `pnpm install`.
3. Start the web app with `pnpm dev:web`.

## Backend

1. Copy `backend/.env.example` to `backend/.env`.
2. Run `make run-api` inside `backend`.

## Supabase

1. Install the Supabase CLI.
2. Run `supabase start`.
3. Apply migrations with `supabase db reset`.

## Mobile

1. Install Flutter.
2. Run `flutter pub get` in `apps/mobile`.
3. Start the app with `flutter run`.
